# esp32 透传 交互协议

## 1.协议格式
```c
typedef struct
{
    uint8_t head;
    uint16_t len;
    uint16_t inf
    uint8_t cmd;
    uint8_t * payload;
    uint8_t foot;
    uint8_t sum;
}

```
**说明**
|名称|说明|
|---|---|
|head|固定头，"L"|
|len|从head到sum|
|inf|保留|
|cmd|详见cmd说明|
|payload|负载内容|
|foot|固定尾，“N”|
|sum|和校验，从head到foot|

## 2.CMD说明

**基于“私有协议规范V0.0.1”**

支持如下命令：
|名称|define|cmd|说明|
|---|---|---|---|
|获取软件版本请求|CMD_VERSION_REQ|0||
|获取软件版本应答|CMD_VERSION_RESP|1||
|获取配置请求|CMD_GET_CONF_REQ|2||
|获取配置应答|CMD_GET_CONF_RESP|3||
|设置配置请求|CMD_SET_CONF_REQ|4||
|设置配置应答|CMD_SET_CONF_RESP|5||
